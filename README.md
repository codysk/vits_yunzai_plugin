# vits_yunzai_plugin

#### 介绍
AI语音本地生成并发送的yunzai插件

#### 软件架构
说明个勾叭


#### 安装教程

1.  yunzai根目录下输入
    git clone https://github.com/sumght-z/vits_yunzai_plugin.git ./
2.  将ys.pth权重文件放入vits/ys/目录下
3.  回到/vits，执行
    pip install -r requirements.txt
    安装环境

#### 使用说明

1.  输入音色列表即可查看可输出角色
2.  输入xxx说xxxx即可生成对应语句
